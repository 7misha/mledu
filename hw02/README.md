# EDA for NYC Airbnb Open Data

Ссылка на датасет:
https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data

Описание:
Датасет содержит объявления Airbnb по Нью-Йорку: характеристики жилья, районы, цены, отзывы, доступность и другие признаки.
Цель работы — провести полный разведочный анализ данных, очистку, предобработку и сделать выводы.
